# Bilibili Mac Client

An unofficial bilibili client for mac

注意：原有百度网盘地址被系统取消掉了，请大家在页面下方使用新的地址下载。

PS: 有点事，下周恢复更新

# Features
- 硬解播放
- libass 显示弹幕，超低 CPU 占用
- 自动拼接分段视频
- 支持发送弹幕
- 弹幕透明度调整
- 本地视频弹幕播放
- 多 Tab 浏览，一键下载视频
- 网页书签调用播放器
- 弹幕关键词屏蔽
- 弹幕智能屏蔽，一键屏蔽吵架，剧透等弹幕
