# Bilibili Mac Client

An unofficial bilibili client for mac

<img src="https://app.ship.io/jobs/LajSkdYLuE7THK7n/build_status.png" style="width:160px" alt="Build Status">

# Features
- 硬解播放
- libass 显示弹幕，超低 CPU 占用
- 自动拼接分段视频
- 支持发送弹幕
- 弹幕透明度调整
- 本地视频弹幕播放
- 多 Tab 浏览，一键下载视频
- 网页书签调用播放器
- 弹幕关键词屏蔽
- 弹幕智能屏蔽，一键屏蔽吵架，剧透等弹幕
